Name

lkb_getselectableboxes - get a list of lockbox handles with trigger conditions that are true

Synopsis

#include <lockbox.h>

int lkb_getselectableboxes(	size_t	arraysize,
				lockbox_t *array);

Description

lkb_getselectableboxes returns a list of lockbox handles which have trigger conditions that are true. These are the lockbox handles that would cause a select(2) to indicate the file descriptor returned by lkb_openvault has an exceptional condition. arraysize is the number of elements in the array pointed to by array. If more than arraysize lockbox handles have trigger conditions that are true, only arraysize lockbox handles will be returned.

Return Value

On success, lkb_setselectcriterion returns the number of lockbox handles returned in array. On failure it returns -1.

Errors

EIO No vault is currently open.
EINTR The call was interrupted by a signal.
EFAULT array is not a valid pointer to arraysize elements.
