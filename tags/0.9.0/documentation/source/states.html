__HEAD__:Accessing State Bits __SEEA__:setstate.html __SEEA__:getstate.html

A process that has a lockbox handle can use it to set the state bits on the lockbox by calling lkb_setstate and retrieve the state bits from the lockbox by calling lkb_getstate.

The state bits are a 32 bit integer. The advantage they have over ordinary data in a lockbox is that state bits can be tested using select(2) by setting appropriate selection criteria on the lockbox handle.
