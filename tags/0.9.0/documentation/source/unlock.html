Name

lkb_unlock - release a prior lock obtained on a lockbox

Synopsis

#include <lockbox.h>

int lkb_unlock(	lockbox_t id);

Description

lkb_unlock releases a lock previously obtained on a lockbox using lkb_lock.

Return Value

On success, lkb_unlock returns 0. On failure, it returns -1.

Errors

EIO No vault is currently open.
ENOENT There is no lockbox open with the handle id.
EINTR The call was interrupted by a signal.
