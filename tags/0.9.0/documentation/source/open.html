Name

lkb_open - open a lockbox on a shelf in the current vault

Synopsis

#include <lockbox.h>

lockbox_t lkb_open(	int shelf,
			char const *name);

Description

lkb_open opens a lockbox on the specified shelf in the current vault. name is the name of the lockbox to open. The lockbox must already exist.

Return Value

On success, lkb_open returns a lockbox handle. On failure, it returns LOCKBOX_ERROR.

Errors

EIO No vault is currently open.
EINVAL name is an empty string.
ENOENT There is no lockbox with that name on the shelf.
EFAULT name does not point to a valid address of a string.
EINTR The call was interrupted by a signal.
EPERM You do not have LKB_ACCESS_OPEN on that lockbox.
