Name

lkb_size - get the size of an open lockbox

Synopsis

#include <lockbox.h>

int lkb_size(lockbox_t id);

Description

lkb_size retrieves the size in bytes of the data stored in the lockbox with the handle id.

Return Value

On success, lkb_size returns 0. On failure, it returns -1.

Errors

EIO No vault is currently open.
ENOENT There is no lockbox open with handle id.
EINTR The call was interrupted by a signal.
EPERM You do not have LKB_ACCESS_READ permission on that lockbox.
