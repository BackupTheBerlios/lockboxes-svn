__HEAD__:lockbox_select_fd_entry __SEEA__:createselectfd.html

Name

lockbox_select_fd_entry - - The structure of an entry in the array passed to lkb_createselectfd

Synopsis

#include <lockbox.h>

typedef struct
{
	uint32_t	lscs_criterion;
	uint32_t	lscs_value;
} lockbox_select_criterion_setting;

typedef struct
{
	lockbox_t	lsfe_id;
	size_t		lsfe_criteria;
	lockbox_select_criterion_setting const *lsfe_settings;
} lockbox_select_fd_entry;

Description

lockbox_select_fd_entry is the structure of an entry in the array passed to lkb_createselectfd. It has the following members:

lsfe_id The lockbox handle of a lockbox opened or created by the current process.
lsfe_criteria The number of criteria to be tested on this lockbox handle.
lsfe_settings A pointer to lsfe_criteria lockbox_select_criterion_setting entries indicating the criteria that will be tested on this lockbox.

lockbox_select_criterion_setting is defined as follows:

lscs_criterion Contains the criterion type as described for the lkb_setselectcriterion call.
lscs_criterion Contains the criterion value as described for the lkb_setselectcriterion call.
