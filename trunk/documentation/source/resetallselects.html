Name

lkb_resetallselects - cancel all trigger conditions on all open lockboxes

Synopsis

#include <lockbox.h>

int lkb_resetallselects(void);

Description

lkb_resetallselects cancels all trigger conditions on all open lockboxes. A subsequent select(2) on the file descriptor returned by lkb_openvault will never indicate that the file descriptor has an exceptional condition.

Return Value

On success, lkb_resetallselects returns 0. On failure it returns -1.

Errors

EIO No vault is currently open.
EINTR The call was interrupted by a signal.
