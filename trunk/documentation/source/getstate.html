Name

lkb_getstate - get the state bits of an open lockbox

Synopsis

#include <lockbox.h>

int lkb_getstate(	lockbox_t id,
			uint32_t *state);

Description

lkb_getstate gets the state bits of the lockbox with the handle id. state a pointer to the location where the state bits should be stored.

Return Value

On success, lkb_getstate returns 0. On failure it returns -1.

Errors

EIO No vault is currently open.
ENOENT There is no lockbox open with handle id.
EINVAL offset is less than 0.
EINTR The call was interrupted by a signal.
EPERM You do not have LKB_ACCESS_GETSTATE permission on that lockbox.
EFAULT state is not a valid address of a uint32_t.
