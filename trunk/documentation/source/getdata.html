Name

lkb_getdata - get data from an open lockbox

Synopsis

#include <lockbox.h>

int lkb_getdata(lockbox_t id,
		void *buffer,
		size_t bufsize,
		off_t offset);

Description

lkb_getdata retrieves data from the lockbox with the handle id. buffer is a pointer to the buffer that will hold the data. bufsize is the number of bytes to read. offset is the offset from the start of the lockbox's data of the first byte to be returned.

Return Value

On success, lkb_getdata returns the number of bytes read, which may be less than bufsize. On failure, it returns -1.

Errors

EIO No vault is currently open.
ENOENT There is no lockbox open with handle id.
EINVAL offset is less than 0.
EINTR The call was interrupted by a signal.
EPERM You do not have LKB_ACCESS_READ permission on that lockbox.
EFAULT buffer is not the valid address of a buffer large enough to hold the data that would be returned.
