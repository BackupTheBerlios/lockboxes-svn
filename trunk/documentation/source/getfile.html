Name

lkb_getfile - get the file from an open lockbox

Synopsis

#include <lockbox.h>

int lkb_getfile(lockbox_t id);

Description

lkb_getfile gets the file from the lockbox with the handle id.

Return Value

On success, lkb_getfile returns a file descriptor for the file. On failure it returns -1.

Errors

EIO No vault is currently open.
ENOENT There is no lockbox open with handle id, or the lockbox does not have a file.
EINTR The call was interrupted by a signal.
EPERM You do not have LKB_ACCESS_GETFD permission on that lockbox.
