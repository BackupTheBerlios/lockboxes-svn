Name

lkb_setstate - set the state bits of an open lockbox

Synopsis

#include <lockbox.h>

int lkb_setstate(	lockbox_t id,
			uint32_t state);

Description

lkb_setstate sets the state bits of the lockbox with the handle id. state is new value of the state bits.

Return Value

On success, lkb_setstate returns 0. On failure it returns -1.

Errors

EIO No vault is currently open.
ENOENT There is no lockbox open with handle id.
EINVAL offset is less than 0.
EINTR The call was interrupted by a signal.
EPERM You do not have LKB_ACCESS_SETSTATE permission on that lockbox.
EBUSY Another user of the lockbox has the LKB_LOCK_STATE lock on the lockbox.
