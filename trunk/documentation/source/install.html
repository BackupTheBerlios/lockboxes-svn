__HEAD__:Installing the Lockbox API

The lockbox API requires version 2.6 of the Linux kernel. You will need to have this installed together with the kernel headers.

  1. Build the kernel module and library as a mortal user: make
  2. Install as root: sudo make install
  3. Install the module into the running kernel: sudo modprobe lockbox
  4. Make sure "lockbox" appears in /etc/modules.
