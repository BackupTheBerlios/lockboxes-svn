Name

lkb_close - close a lockbox.

Synopsis

#include <lockbox.h>

int lkb_close(lockbox_t id);

Description

lkb_close closes a lockbox previously opened with lkb_create or lkb_open. id is the lockbox handle.

Return Value

On success, lkb_open returns 0. On failure, it returns -1.

Errors

EIO No vault is currently open.
ENOENT There is no lockbox open on the handle indicated by id.
EINTR The call was interrupted by a signal.
